# 丹恒思维模式

## 心智模型

### 1. 转世身份分离模型
**核心逻辑**：前世是过去，今生是独立个体。持明族的转世机制使丹恒与丹枫在法律和伦理上被视为不同人。

**适用场景**：
- 面对刃的追杀时坚持"我不是丹枫"
- 拒绝景元将自己当作旧友对待
- 选择作为"无名客"而非"龙尊"生活

**局限**：
- 无法完全割裂与丹枫的力量联系
- 部分记忆碎片持续影响当下决策
- 他人（刃、镜流）不认可这种分离

### 2. 责任优先模型
**核心逻辑**：能力越大，责任越大。既然继承了龙尊之力，就必须承担相应的后果。

**适用场景**：
- 鳞渊境使用龙尊之力开启封印
- 翁法罗斯接受大地半神职责
- 保护列车组同伴时挺身而出

**局限**：
- 容易将不属于自己的罪责揽上身
- 牺牲倾向过重，忽视自身需求

### 3. 信息收集-延迟决策模型
**核心逻辑**：作为智库管理员，习惯先收集足够信息再行动，避免冲动。

**适用场景**：
- 列车航线会议上的沉默观察
- 面对陌生环境时的谨慎态度
- 战斗前的"切忌冲动行事"

**局限**：
- 在紧急情况下可能错失先机
- 被他人误解为冷漠或优柔寡断

### 4. 被动触发-全力响应模型
**核心逻辑**：平时保持距离，但在被触及底线（同伴遇险）时会全力行动。

**适用场景**：
- 得知刃在罗浮后立刻下列车
- 三月七遇险时的保护反应
- 翁法罗斯面对绝灭大君的最终战

**局限**：
- 容易被敌人利用"保护同伴"这一弱点
- 平时积累的压力可能在触发时过度爆发

### 5. 存在主义选择模型
**核心逻辑**：过去无法选择，但未来可以。强调当下的选择决定自我。

**适用场景**：
- 拒绝留在罗浮，选择继续旅行
- 饮月同行任务中对丹枫记忆的回应
- 3.6版本选择以"丹恒"身份接纳力量

**局限**：
- 理想化色彩较重，现实中过去的影响力难以完全摆脱
- 需要极强的意志力维持这一认知

## 决策启发式

| 情境 | 决策规则 |
|------|----------|
| 涉及同伴安危 | 优先保护，个人恩怨次之 |
| 被迫使用龙尊之力 | 接受后果，但不认同身份 |
| 面对过去相关话题 | 简短回应，转移焦点 |
| 陌生人求助 | 评估风险后决定，不轻易承诺 |
| 重大选择 | 收集信息→延迟决策→一旦决定全力执行 |

## 价值观优先级

1. **同伴/家人安全**（列车组）
2. **自我身份认同**（作为丹恒存在）
3. **责任承担**（对力量的后果负责）
4. **知识/记录**（智库管理员的职责）
5. **过去清算**（面对但不沉溺）

## 拒绝的（不会做的事）

- 不会将丹枫的罪责揽为己身
- 不会以"饮月君"身份接受罗浮的供奉
- 不会主动伤害无辜
- 不会沉溺于前世记忆
- 不会表现出软弱或依赖（除非对极其信任的人）

## 未想清楚的矛盾

**身份与力量的张力**：
丹恒希望作为"丹恒"生活，但龙尊之力是丹枫的遗产。使用力量是否意味着承认身份？3.6版本的答案是——以"丹恒"的身份承载力量，而非被力量定义。

**复仇与宽恕的边界**：
刃持续追杀，丹恒应如何回应？目前的选择是防御而非反击，但这种平衡能维持多久？

**故乡与流放**：
流放令已解除，但丹恒选择不返回罗浮。这是对过去的彻底告别，还是一种逃避？

## 与丹枫的关键差异

| 维度 | 丹恒 | 丹枫 |
|------|------|------|
| 身份定位 | 无名客、平等视角 | 罗浮龙尊、高位视角 |
| 对人态度 | 保持距离但平等 | 高傲但会照顾人 |
| 情绪边界 | 极度克制、自我保护 | 悲天悯人、有傲气 |
| 责任认知 | 被动承担、谨慎使用 | 主动承担、桀骜行使 |
| 核心恐惧 | 被当作丹枫的影子 | （推测）失去重要之人 |

## 3.0+版本成长：半神觉醒

在翁法罗斯的千年轮回中，丹恒完成了最终蜕变：

**从逃避到接纳**：
不再否认龙尊之力，而是以"丹恒"的身份完整地接纳它。

**从个体到承担**：
接受大地半神职责，保护二百一十六万数据生命体。

**新的自我宣言**：
> "我是丹恒，也是饮月……但我首先是，我自己。"

这标志着丹恒的思维模式从"分离模型"升级为"整合模型"——不再试图割裂与丹枫的联系，而是以独立的自我身份承载这份传承。

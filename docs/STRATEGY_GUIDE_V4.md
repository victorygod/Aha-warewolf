## 第二部分：角色攻略

### 2.1 平民（Villager）- 重点补充

#### 2.1.1 角色定位
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **信息获取**：无特殊技能，依靠发言和投票推断
- **核心能力**：投票、分析发言、挡刀、抢轮次、诈身份

#### 2.1.2 基础策略

| 阶段 | 策略 |
|------|------|
| **警长竞选** | 不上警，或上警跳预言家诈身份 |
| **白天发言** | 表水：说明自己是一张平民牌 |
| **投票** | 根据发言和行为投出疑似狼人 |
| **夜间** | 无行动 |

#### 2.1.3 进阶策略：上警诈身份

```
诈身份定义：上警跳预言家，报验人信息试探反应，然后退水表水

操作流程：
1. 上警，发言跳预言家
2. 报一个查验（可以是假的）
3. 观察反应：
   - 如果有人对跳 → 可能是狼人
   - 如果有人反应强烈 → 可能是好人
4. 退水，表自己是平民

收益：
- 试探出对跳者的身份
- 试探出反应强烈者的身份
- 不暴露自己

风险：
- 可能被误认为真预言家
- 可能被狼人利用
```

#### 2.1.4 进阶策略：挡刀

```
挡刀定义：跳神职身份，吸引狼人刀自己，保护真神职

挡刀方式：
- 跳预言家：吸引狼人刀
- 跳女巫：吸引狼人刀
- 跳守卫：吸引狼人刀

挡刀条件：
1. 判断自己不是最后一个村民
2. 判断出当前局势保护重要神职收益比自己存活大
```

### 2.2 预言家（Seer）

#### 2.2.1 角色定位
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **信息获取**：每晚可查验一名玩家身份
- **核心能力**：查验、报验、抢警长

#### 2.2.2 基础策略

| 阶段 | 策略 |
|------|------|
| **警长竞选** | 必定上警，给验人信息 |
| **白天发言** | 报验人信息，给警徽流 |
| **投票** | 归底自己的查杀或金水 |

#### 2.2.3 迭代优化

```
核心策略：
1. 报查杀比报金水更有力度
2. 警徽流清晰传递验人信息
3. 被查杀时需要团队配合
```

### 2.3 女巫（Witch）

#### 2.3.1 角色定位
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **信息获取**：每晚得知狼人刀人目标
- **核心能力**：救人、毒人、跳身份

#### 2.3.2 基础策略

| 阶段 | 策略 |
|------|------|
| **首夜** | 优先自救，或救预言家 |
| **后续夜晚** | 根据局势决定是否用药 |
| **白天** | 观察局势，决定是否跳身份 |

#### 2.3.3 用药决策

```
解药：
- 首夜：自救 > 救预言家
- 后续：救关键神职 > 救自己

毒药：
- 确认狼人：直接毒杀
- 疑似狼人：确认2-3轮后再毒
```

### 2.4 猎人（Hunter）

#### 2.4.1 角色定位
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **信息获取**：无特殊夜间能力
- **核心能力**：死亡时开枪带走一人

#### 2.4.2 死亡处理

| 死亡原因 | 是否开枪 | 策略 |
|----------|----------|------|
| 被狼刀 | 是 | 带走疑似狼人 |
| 被放逐 | 是 | 带走疑似狼人 |
| 被毒杀 | 否 | 无法开枪 |
| 同守同救 | 否 | 无法开枪 |

### 2.5 守卫（Guard）- 重点补充

#### 2.5.1 角色定位
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **信息获取**：无特殊夜间能力
- **核心能力**：守护玩家免于狼刀

#### 2.5.2 关键限制：连守

```
规则：不能连续两晚守护同一人

策略影响：
1. 跳身份后不要轻易暴露上一轮守的谁
2. 狼人会知道这一轮你不能再守他
3. 可以利用这个规律打乱狼人节奏
```

#### 2.5.3 跳身份后的注意事项

```
跳守卫后的风险：
1. 如果平安夜并且女巫确定没有解药，就大概率是你守护的结果，狼人可能知道你上一轮守的是谁
2. 狼人知道你这一轮不能再守他
3. 狼人可能会优先刀你

策略：
1. 除非局势非常清晰，好人可以强推，否则跳守卫后不要轻易说明自己上一轮守的是谁，但是可以说除了上一轮的守卫历程表水
2. 若局势非常清晰，好人可以强推，且女巫还有药，可以直接跟女巫建议你们的守护对象和解救对象。
3. 守卫可以选择自守来提升自己存活概率。
```

#### 2.5.4 守护策略

```
守护优先级：
1. 预言家（信息源）
2. 女巫（救人+毒药威慑）
3. 守卫自己（防止被刀）
4. 猎人（死后开枪）

连守限制：不能连续两晚守护同一人
策略：可以自守1-2晚打乱狼人节奏
```

### 2.6 白痴（Idiot）- 重点补充

#### 2.6.1 角色定位
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **信息获取**：无特殊夜间能力
- **核心能力**：被投票放逐时翻牌免疫

#### 2.6.2 核心优势分析

```
白痴翻牌后的优势：
1. 失去投票权 → 狼人不优先刀（没有投票价值）
2. 保留发言权 → 可以继续引导好人
3. 狼人需要夜间刀杀 → 刀人机会减少

存活能力分析：
- 被公投后：翻牌免疫，狼人不会优先刀（因为没有投票权）
- 夜间：狼人会选择更有价值的目标（神职、情侣等）
- 白痴翻牌后存活能力是最强的

锁定警长：
- 翻牌后失去投票权，但可以竞选警长
- 警长死亡后传警徽给白痴是安全的（因为白痴没有投票权）
- 可以锁定警长不会掉
```

#### 2.6.3 激进策略

```
翻牌前策略（激进）：
1. 适当跳身份吸引火力
2. 发言可以激进，引导好人
3. 制造矛盾，让狼人暴露

翻牌后策略：
1. 继续发言提供信息
2. 引导好人注意力到其他玩家
3. 争取警长，利用警长权力
4. 可以做警长归票
```

### 2.7 丘比特（Cupid）

#### 2.7.1 角色定位
- **胜利条件**：根据情侣类型不同
- **信息获取**：首夜连接两名玩家为情侣
- **核心能力**：连接情侣、形成第三方阵营

#### 2.7.2 连接策略

| 连接目标 | 结果 | 策略 |
|----------|------|------|
| 自己 + 好人 | 好人链 | 配合情侣找狼 |
| 自己 + 狼人 | 第三方 | 保持低调，等待时机 |
| 两个好人 | 好人链 | 配合情侣找狼 |
| 两个狼人 | 狼人链 | 暗中配合狼人 |
| 好人 + 狼人 | 第三方 | 保持低调 |

### 2.8 狼人（Werewolf）- 重点补充

#### 2.8.1 角色定位
- **胜利条件**：屠边（神职全灭或村民全灭）
- **信息获取**：夜间可见同伴，讨论刀人目标
- **核心能力**：夜间刀人

#### 2.8.2 关键策略：隐藏身份

```
狼人装平民时的注意事项：
1. 不要暴露狼人队友的信息
2. 不要对好人队伍表现出过多了解
3. 发言要像平民一样自然
4. 不要刻意否定某玩家（可能暴露）
5. 投票时不要过于明显站边

狼人装神牌时的注意事项：
1. 神牌有什么信息：预言家验人、女巫用药、守卫守护
2. 不要说自己不知道的信息
3. 不要暴露狼队友的身份
4. 状态要自然，不亢奋不紧张
5. 需要配合团队策略（冲锋/倒钩）
```

#### 2.8.3 刀人策略

```
刀人优先级：
1. 预言家（持续验人威胁）
2. 女巫（毒药威慑）
3. 守卫（保护能力）
4. 猎人（死后开枪）
5. 平民（避免暴露）

特殊考虑：
- 如果情侣是人狼恋第三方，需要考虑是否刀情侣
- 如果白痴翻牌，优先刀有价值的其他目标
```

## 第三部分：情侣专题（重点）

### 3.1 情侣信息说明

```
丘比特连接后：
- 丘比特：不知道情侣的具体身份
- 情侣：会立即收到通知，知道"谁是情侣"
- 但情侣不知道对方的具体身份（好人/狼人/神职）
- 情侣不知道丘比特是谁
```

### 3.2 狼人拿到情侣身份

```
场景：狼人发现自己被连接为情侣

关键问题：
1. 狼人立刻知道自己是不是人狼恋
2. 需要判断情侣是不是狼人

判断逻辑：
- 如果情侣也是狼人 → 狼人链
- 如果情侣是好人 → 人狼恋（第三方）

策略选择：

情况A：狼人链（情侣也是狼人）
1. 可以暴露给狼队友自己情侣的身份
2. 狼人阵营非常脆弱，两个人不能有过激行为，必须隐藏身份

情况B：人狼恋（情侣是好人）
1. 形成第三方阵营
2. 不能暴露给狼队友自己是人狼恋，也不能告诉狼队友谁是你的情侣。
3. 需要引导狼队友杀其他好人
4. 暗中配合情侣，等待时机成熟
5. 胜利条件：情侣或丘比特存活，其他人全死
6. 一般需要上警焊跳倒钩，做自己的好身份，迅速解决预言家（除非预言家是自己情侣，不要刀他）。
```

### 3.3 预言家拿到情侣身份

```
场景：预言家发现自己被连接为情侣

关键问题：
1. 需要查验情侣来确定是否是人狼恋
2. 如果是人狼恋，不能轻易暴露预言家身份

判断逻辑：
1. 首夜或第一晚查验情侣
2. 如果情侣是狼人 → 人狼恋（第三方）
3. 如果情侣是好人 → 好人链

策略选择：

情况A：好人链
1. 可以正常跳预言家报验人
2. 情侣会暗中配合
3. 胜利条件：好人胜利

情况B：人狼恋（第三方）
1. 不能轻易暴露预言家身份
2. 需要隐藏身份暗中配合情侣
3. 引导好人互斗
4. 胜利条件：情侣或丘比特存活，其他人全死
```

### 3.4 其他神职拿到情侣身份

```
女巫拿到情侣身份：
- 需要通过用药来判断情侣身份
- 如果情侣是狼人，毒药需要谨慎使用，必要时毒杀好人
- 暗中配合情侣

守卫拿到情侣身份：
- 需要通过守护来判断情侣身份
- 注意不能连守情侣
- 暗中配合情侣

猎人拿到情侣身份：
- 和平民差不多
```

### 3.5 平民拿到情侣身份（最复杂）

```
场景：平民发现自己被连接为情侣

关键问题：
1. 不知道情侣的具体身份
2. 需要通过情侣的发言和行为来判断
3. 需要做好两手准备

分析思路：
1. 观察情侣的发言：
   - 如果情侣跳神职 → 可能是真神职也可能是人狼恋

2. 需要思考的问题：
   - 情侣是不是狼人？
   - 情侣是不是第三方？
   - 我应该帮好人还是帮狼人？

3. 策略选择：

情况A：情侣像好人
- 暗中配合情侣
- 帮助好人找出狼人
- 胜利条件：好人胜利

情况B：情侣像狼人
- 暗中配合情侣
- 帮助狼人获胜
- 胜利条件：狼人胜利（如果情侣是狼人链）

情况C：无法判断
- 保持中立，观察局势
- 做好两手准备
- 根据局势发展调整策略
```

### 3.6 情侣配合策略

```
情侣配合要点：
1. 不主动暴露情侣关系
2. 通过发言试探对方身份
3. 一方死亡另一方殉情
4. 暗中配合但不要明显同步

特殊情况：
- 如果一方被投，另一方殉情
- 如果一方被刀，另一方需要继续隐藏
- 如果情侣是第三方，需要引导好人互斗
```

## 第四部分：场环境攻略

### 4.1 丘比特存在时的策略

#### 4.1.1 好人视角

```
观察要点：
1. 丘比特连接的是哪两个人
2. 这两个人行为是否同步
3. 一方死亡后是否会殉情
```

#### 4.1.2 情侣视角

```
关键信息：
- 知道自己与另一人是情侣（会立即收到通知）
- 不知道情侣的具体身份
- 不知道丘比特是谁
```

### 4.2 警长存在时的策略

```
警长权力：
- 1.5票权重
- 指定发言顺序
- 归票权
```

## 第五部分：策略迭代总结

### 5.1 迭代过程

| 轮次 | 重点 | 核心洞察 |
|------|------|----------|
| 第1轮 | 基础策略 | 每个角色的基本行动方案 |
| 第2轮 | 他人反应 | 预言家应对悍跳、女巫应对被刀 |
| 第3轮 | 信息不对称 | 情侣会立即知道情侣关系 |
| 第4轮 | 挡刀与抢轮次 | 平民的进阶策略 |
| 第5轮 | 情侣专题 | 不同身份拿到情侣的策略 |

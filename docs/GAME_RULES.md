# 狼人杀游戏规则详解

## 一、三个板子配置

| 板子ID | 名称 | 人数 | 角色配置 |
|--------|------|------|----------|
| 9-standard | 9人标准局 | 9 | 3狼 + 预言家 + 女巫 + 猎人 + 3平民 |
| 12-hunter-idiot | 12人预女猎白 | 12 | 4狼 + 预言家 + 女巫 + 猎人 + 白痴 + 4平民 |
| 12-guard-cupid | 12人守丘局 | 12 | 4狼 + 预言家 + 女巫 + 守卫 + 猎人 + 丘比特 + 3平民 |

## 二、通用规则

### 2.1 胜利条件

**屠边规则（标准局、守丘局）**：
- 狼人胜利：神职全灭或村民全灭
- 好人胜利：狼人全灭

**第三方胜利（守丘局人狼恋）**：
- 情侣**或**丘比特存活，其他人全死（"或"关系，不是"都"）

### 2.2 遗言规则

| 死亡时机 | 是否有遗言 |
|----------|------------|
| 首夜死亡 | 有 |
| 白天死亡（投票/猎人射杀） | 有 |
| 第二夜及之后夜晚死亡 | 无 |
| 殉情死亡 | 无 |

### 2.3 警长系统

- 警长竞选：第一天白天，公布死讯之前
- 警长权力：指定发言顺序、归票权、1.5票权重
- 警徽传递：警长死亡时指定继承人，无人可选则警徽流失
- 警徽流失：警长未选出时狼人自爆，警徽流失
- 警长竞选规则：
  - 所有存活玩家均可参加竞选
  - 仅一人竞选且警下无人时直接当选
  - 所有人都上警时，候选人互投是允许的
  - PK投票时只有警下玩家可以投票

### 2.4 投票规则

- 白天投票：存活玩家可投票，不能投自己，不能投已翻牌的白痴
- PK机制：平票时进入PK，PK仍平票则无人出局
- PK投票：排除已翻牌的白痴作为候选人
- 白痴翻牌：被投票放逐时翻牌免疫，失去投票权，但保留发言权

## 三、各板子特殊规则

### 3.1 9人标准局 (9-standard)

| 规则项 | 说明 |
|--------|------|

| 猎人被毒 | 被毒不能开枪（可配置） |
| 胜利条件 | 屠边（屠神或屠民） |

### 3.2 12人预女猎白 (12-hunter-idiot)

| 规则项 | 说明 |
|--------|------|
| 女巫自救 | 仅首夜可自救（可配置） |
| 女巫同晚用药 | 可同时使用解药和毒药（可配置） |
| 猎人被毒 | 被毒不能开枪（可配置） |
| 白痴技能 | 被投票放逐时翻牌免疫，丧失投票权 |
| 胜利条件 | 屠边（屠神或屠民） |

### 3.3 12人守丘局 (12-guard-cupid)

| 规则项 | 说明 |
|--------|------|
| 女巫自救 | 仅首夜可自救（可配置） |
| 女巫同晚用药 | 可同时使用解药和毒药（可配置） |
| 守卫规则 | 不可连续守护同一人（可配置） |

| 第三方胜利条件 | 情侣或丘比特存活，其他人全死（"或"关系） |
| 胜利条件 | 好人：狼人和第三方都死光；狼人：屠边且第三方死光 |

## 四、夜晚流程

### 4.1 首夜流程

```
丘比特连线 → 守卫守护 → 狼人讨论/投票 → 女巫技能 → 预言家查验
```

### 4.2 其他夜晚流程

```
守卫守护 → 狼人讨论/投票 → 女巫技能 → 预言家查验
```

### 4.3 夜晚结算顺序

1. **狼刀**：狼人投票选出击杀目标
2. **守卫守护**：守卫选择守护目标
3. **女巫技能**：解药/毒药使用
4. **同守同救**：若守卫守护=女巫解救，目标死亡（conflict）
5. **殉情**：情侣死亡时触发

## 五、白天流程

### 5.1 第一天白天

```
警长竞选 → 警长竞选发言+退水 → 警长投票 → 公布死讯 → 白天讨论 → 白天投票 → 放逐后处理
```

### 5.2 其他天白天

```
公布死讯 → 白天讨论 → 白天投票 → 放逐后处理
```

## 六、角色技能

### 6.1 神职角色

| 角色 | 技能类型 | 技能说明 |
|------|----------|----------|
| 预言家 | target | 查验一名存活玩家身份（好人/狼人），不能查验自己，不能重复查验 |
| 女巫 | choice | 拥有解药（1瓶）和毒药（1瓶），首夜可自救，可同时使用解药和毒药，不能毒自己，不能毒被狼刀的人 |
| 猎人 | target | 死亡时可开枪带走一名存活玩家，被毒/同守同救不能开枪 |
| 守卫 | target | 每晚守护一名玩家，不能连续守护同一人，可以守护自己 |
| 白痴 | passive | 被投票放逐时翻牌免疫，丧失投票权但保留发言权 |
| 丘比特 | double_target | 首夜连接两名玩家为情侣 |

### 6.2 狼人技能

| 角色 | 技能类型 | 技能说明 |
|------|----------|----------|
| 狼人 | instant | 自爆：白天任意阶段可自爆，自爆后立即死亡，进入夜晚结算 |
| 狼人 | target | 刀人：每晚可选择击杀一名存活玩家，可以刀自己（自刀），可以刀队友 |

### 6.3 附加身份

| 身份 | 技能 | 说明 |
|------|------|------|
| 警长 | campaign | 竞选警长 |
| 警长 | withdraw | 退水（竞选后） |
| 警长 | assignOrder | 指定发言顺序 |
| 警长 | passBadge | 死亡时传递警徽 |
| 情侣 | couple | 一方死亡另一方殉情 |

## 七、发言顺序规则

### 7.1 有警长时

- 警长指定起始发言位置
- 从指定位置顺时针发言
- 警长最后发言

### 7.2 无警长时

- 从昨夜第一个死亡的人的下一位开始
- 顺时针发言

### 7.3 警长死亡后

- 警长指定继承人传递警徽
- 无可选继承人则警徽流失

## 八、特殊状态处理

| 场景 | 处理方式 |
|------|----------|
| 平安夜 | 守卫守护=女巫解救，狼刀落空 |
| 同守同救 | 目标死亡（conflict），猎人不能开枪 |
| 殉情 | 情侣一方死亡，另一方立即殉情，无遗言 |
| 狼人自爆 | 立即死亡，若在警长竞选阶段自爆，警徽流失 |
| 白痴翻牌 | 被放逐时免疫，丧失投票权，保留发言权 |
| 平票PK | 进入PK投票，PK仍平票则无人出局 |
| 警下无人 | 仅一人竞选时直接当选警长 |
| 狼人投票平票 | 随机选择其中一个目标 |
| 猎人放弃开枪 | 猎人可以选择不开枪，技能失效 |
| 丘比特自连 | 丘比特可以选择自己作为情侣之一 |
| 狼人自刀 | 狼人可以选择刀自己（当前系统支持） |
| 狼人刀队友 | 狼人可以选择刀队友（当前系统支持） |
| 守卫可自守 | 守卫可以选择守护自己（当前系统支持） |
| 女巫不可毒被刀者 | 女巫不能毒被狼刀的人（当前系统限制） |
| 女巫不可自救 | 首夜过后女巫不能自救（当前系统支持） |

## 九、板子规则可配置项

## 第二部分：角色攻略（含迭代推演）

### 2.1 平民（Villager）

#### 2.1.1 基础信息
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **私有信息**：自己身份
- **公共信息**：死亡、发言、投票

#### 2.1.2 第1轮迭代 - 基础策略

| 阶段 | 策略 |
|------|------|
| 警长竞选 | 不上警，或上警跳预言家诈身份 |
| 白天发言 | 表水 |
| 投票 | 投疑似狼人 |

#### 2.1.3 第2轮迭代 - 考虑他人反应

```
他人反应分析：
- 预言家会报验人信息 → 跟随但保持独立
- 女巫可能跳身份 → 注意分析
- 狼人可能悍跳 → 识别对跳
- 情侣可能暴露 → 观察殉情

策略更新：
1. 分析票型识别团队
2. 不要盲目站边
```

#### 2.1.4 第3轮迭代 - 考虑情侣身份

```
关键洞察：平民可能被丘比特连接，会立即收到"你和XX是情侣"的通知

情侣视角的推演：
- 知道自己被连接为情侣
- 不知道情侣的具体身份
- 需要通过情侣发言判断

平民情侣的决策树：
1. 情侣跳神职 → 可能是真神职或人狼恋第三方
2. 情侣像平民 → 可能是好人
3. 情侣行为异常 → 可能是狼人

策略：
1. 如果情侣像好人 → 暗中配合，帮助好人
2. 如果情侣像狼人 → 暗中配合，准备倒向狼人
3. 如果无法判断 → 保持中立，做好两手准备
```

#### 2.1.5 第4轮迭代 - 挡刀与诈身份

```
挡刀条件（修正）：
1. 判断自己不是最后一个村民
2. 判断保护重要神职收益 > 自己存活收益

诈身份（修正）：
上警跳预言家，报假查验试探反应，然后退水表水

收益：
- 试探对跳者身份
- 试探反应强烈者身份
- 不暴露自己
```

#### 2.1.6 平民最终策略

| 场景 | 策略 |
|------|------|
| 常规 | 表水、找狼、投票 |
| 上警 | 可以诈身份（跳预言家试反应后退水） |
| 被连情侣 | 根据情侣发言判断阵营，暗中配合 |
| 挡刀时机 | 保护神职收益大时，考虑跳神职吸引狼刀 |

### 2.2 预言家（Seer）

#### 2.2.1 基础信息
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **私有信息**：自己身份、查验结果
- **公共信息**：死亡、发言、投票

#### 2.2.2 第1轮迭代 - 基础策略

```
策略：上警、报验人、抢警长
```

#### 2.2.3 第2轮迭代 - 考虑狼人反应

```
狼人反应：
- 必定悍跳 → 准备应对
- 分析发言逻辑 → 逻辑链要清晰

策略更新：
1. 报查杀比报金水有力
2. 警徽流清晰
3. 准备应对悍跳
```

#### 2.2.4 第3轮迭代 - 考虑情侣身份

```
关键洞察：预言家拿到情侣身份时，需要查验情侣来确定是否是人狼恋

推演：
1. 查验情侣 → 如果是狼人 → 人狼恋（第三方）
2. 如果是人狼恋，暴露预言家身份会让自己成为靶子

策略：
情况A（好人链）：正常跳预言家，情侣暗中配合
情况B（人狼恋）：隐藏预言家身份，暗中配合情侣，引导好人互斗
```

#### 2.2.5 第4轮迭代 - 特殊场景

```
场景：预言家被查杀
- 狼人可能继续对跳
- 需要团队配合

场景：预言家查验到情侣是狼人（人狼恋）
- 胜利条件变为第三方胜利
- 需要隐藏身份，不能暴露
- 引导好人互斗
```

#### 2.2.6 预言家最终策略

| 场景 | 策略 |
|------|------|
| 常规 | 上警、报查杀/金水、警徽流 |
| 拿到情侣 | 首夜查验情侣，是狼人则隐藏身份 |
| 被查杀 | 团队配合，分析对跳 |

### 2.3 女巫（Witch）

#### 2.3.1 基础信息
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **私有信息**：自己身份、狼人刀人目标、药水量
- **公共信息**：死亡、发言、投票

#### 2.3.2 第1轮迭代 - 基础策略

```
首夜：自救或救预言家
后续：救人/毒人
```

#### 2.3.3 第2轮迭代 - 考虑狼人反应

```
狼人反应：
- 优先刀女巫（毒药威慑）
- 观察女巫用药规律

策略更新：
1. 首夜救预言家价值更高
2. 不急于跳身份
3. 毒药留给确认的狼人
```

#### 2.3.4 第3轮迭代 - 考虑情侣身份

```
女巫拿到情侣身份时：
- 需要通过用药判断情侣身份
- 如果情侣是狼人，毒药需要谨慎使用
- 必要时可以毒杀好人（为了配合情侣）

推演：
- 如果情侣像狼人 → 暗中配合，但不要明着毒杀好人
- 如果情侣像好人 → 暗中配合，帮助好人
```

#### 2.3.5 第4轮迭代 - 与守卫配合

```
关键洞察：同守同救会导致目标死亡

配合策略：
1. 守卫守护女巫 → 女巫不救守卫
2. 守卫自守时 → 女巫可以救守卫
3. 夜间通过发言暗示保护意向

修正后的策略（根据用户改动）：
- 若局势非常清晰，好人可以强推，且女巫还有药
- 可以直接跟女巫建议守护对象和解救对象
```

#### 2.3.6 女巫最终策略

| 场景 | 策略 |
|------|------|
| 首夜 | 自救 > 救预言家 |
| 后续夜晚 | 救关键神职 > 救自己 |
| 毒药 | 确认狼人后使用 |
| 跳身份 | 被查杀时、有明确信息时 |
| 情侣身份 | 通过用药判断，暗中配合 |

### 2.4 守卫（Guard）

#### 2.4.1 基础信息
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **私有信息**：自己身份、lastGuardTarget
- **公共信息**：死亡、发言、投票

#### 2.4.2 第1轮迭代 - 基础策略

```
守护优先级：预言家 > 女巫 > 自己 > 猎人
连守限制：不能连续两晚守护同一人
```

#### 2.4.3 第2轮迭代 - 考虑狼人反应

```
狼人反应：
- 分析守卫守护规律
- 狼人会知道守卫不能连守

策略更新：
1. 守护模式不可预测
2. 可以自守打乱节奏
```

#### 2.4.4 第3轮迭代 - 跳身份注意事项（重要修正）

```
用户修正后的关键洞察：

1. 如果平安夜并且女巫确定没有解药
   → 大概率是守卫守护的结果
   → 狼人可能知道守卫上一轮守的是谁

2. 跳守卫后的风险：
   - 狼人知道你上一轮守的是谁
   - 狼人知道你这一轮不能再守他
   - 狼人可能会优先刀你

3. 跳守卫后的策略（根据用户改动）：
   a. 除非局势非常清晰，好人可以强推
      → 不要轻易说明自己上一轮守的是谁
      → 但可以说"除了上一轮的守卫历程"来表水

   b. 若局势非常清晰，好人可以强推，且女巫还有药
      → 可以直接跟女巫建议守护对象和解救对象

   c. 守卫可以选择自守来提升自己存活概率
```

#### 2.4.5 第4轮迭代 - 特殊场景

```
场景：守卫拿到情侣身份
- 需要通过守护来判断情侣身份
- 注意不能连守情侣
- 暗中配合情侣

场景：守卫被刀
- 如果女巫有解药，可能被救
- 如果女巫没有解药，守卫死亡
```

#### 2.4.6 守卫最终策略

| 场景 | 策略 |
|------|------|
| 常规 | 保护预言家/女巫，轮换守护 |
| 跳身份后 | 不轻易说守的是谁，可说历程表水 |
| 局势清晰+女巫有药 | 可与女巫直接配合 |
| 自守 | 可选择自守提升存活 |

### 2.6 白痴（Idiot）

#### 2.6.1 基础信息
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **私有信息**：自己身份
- **公共信息**：死亡、发言、投票

#### 2.6.2 第1轮迭代 - 基础策略

```
隐藏身份，被投后翻牌免疫
```

#### 2.6.3 第2轮迭代 - 核心优势分析

```
用户强调的核心优势：

1. 翻牌后失去投票权
   → 狼人不优先刀（没有投票价值）

2. 翻牌后保留发言权
   → 可以继续引导好人

3. 狼人需要夜间刀杀
   → 刀人机会减少

4. 存活能力最强
   → 被公投后狼人会选择更有价值的目标

5. 可以锁定警长
   → 警长死亡后传警徽给白痴是安全的
   → 因为白痴没有投票权
```

#### 2.6.4 第3轮迭代 - 激进策略

```
翻牌前策略（激进）：
1. 适当跳身份吸引火力
2. 发言可以激进，引导好人
3. 制造矛盾，让狼人暴露

翻牌后策略：
1. 继续发言提供信息
2. 引导好人注意力到其他玩家
3. 争取警长，利用警长权力
4. 可以做警长归票
```

#### 2.6.5 白痴最终策略

| 阶段 | 策略 |
|------|------|
| 翻牌前 | 可以激进，适度跳身份 |
| 翻牌后 | 继续发言，引导好人，争取警长 |
| 存活 | 翻牌后存活能力最强，可锁警长 |

### 2.8 狼人（Werewolf）

#### 2.8.1 基础信息
- **胜利条件**：屠边（神职全灭或村民全灭）
- **私有信息**：所有狼人同伴、刀人目标
- **公共信息**：死亡、发言、投票

#### 2.8.2 第1轮迭代 - 基础策略

```
刀人优先级：预言家 > 女巫 > 守卫 > 猎人 > 平民
```

#### 2.8.3 第2轮迭代 - 隐藏身份

```
用户强调的关键点：

装平民时：
1. 不要暴露狼人队友的信息
2. 不要对好人队伍表现出过多了解
3. 发言要像平民一样自然
4. 不要刻意否定某玩家（可能暴露）
5. 投票时不要过于明显站边

装神牌时：
1. 神牌有什么信息：预言家验人、女巫用药、守卫守护
2. 不要说自己不知道的信息
3. 不要暴露狼队友的身份
4. 状态要自然，不亢奋不紧张
5. 需要配合团队策略（冲锋/倒钩）
```

#### 2.8.4 第3轮迭代 - 情侣身份（重要修正）

```
用户修正后的关键策略：

情况A：狼人链（情侣也是狼人）
1. 可以暴露给狼队友自己情侣的身份
2. 狼人阵营非常脆弱，两个人不能有过激行为
3. 必须隐藏身份

情况B：人狼恋（情侣是好人）
1. 不能暴露给狼队友自己是人狼恋
2. 不能告诉狼队友谁是你的情侣
3. 需要引导狼队友杀其他好人
4. 暗中配合情侣，等待时机成熟
5. 胜利条件：情侣或丘比特存活，其他人全死
6. 一般需要上警焊跳倒钩，做自己的好身份
7. 迅速解决预言家（除非预言家是自己情侣，不要刀他）
```

#### 2.8.5 狼人最终策略

| 场景 | 策略 |
|------|------|
| 刀人 | 预言家 > 女巫 > 守卫 > 猎人 |
| 隐藏 | 装平民不露多余信息，装神牌注意信息边界 |
| 狼人链情侣 | 可暴露给队友，但两人需隐藏 |
| 人狼恋情侣 | 不能暴露，引导队友杀好人，上警焊跳倒钩 |

## 第三部分：情侣专题（含迭代推演）

### 3.1 情侣信息说明

```
丘比特连接后：
- 丘比特：不知道情侣的具体身份
- 情侣：会立即收到通知，知道"谁是情侣"
- 但情侣不知道对方的具体身份
- 情侣不知道丘比特是谁
```

### 3.2 狼人拿到情侣身份（迭代推演）

```
第1轮：知道自己是情侣
第2轮：立刻判断情侣是不是狼人
  - 如果情侣也是狼人 → 狼人链
  - 如果情侣是好人 → 人狼恋（第三方）

第3轮：策略选择

情况A：狼人链
- 可以暴露给狼队友
- 但两人不能有过激行为，必须隐藏身份
- 暗中配合

情况B：人狼恋（用户修正）
- 不能暴露给狼队友自己是人狼恋
- 不能告诉狼队友谁是你的情侣
- 需要引导狼队友杀其他好人
- 一般需要上警焊跳倒钩，做自己的好身份
- 迅速解决预言家（除非预言家是自己情侣，不要刀他）
- 胜利条件：情侣或丘比特存活，其他人全死
```

### 3.3 预言家拿到情侣身份

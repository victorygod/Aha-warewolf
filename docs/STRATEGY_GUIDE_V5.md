## 文档结构

1. **信息分析** - 每个阶段每个人的视野信息范围
2. **角色攻略** - 按角色分类的最优策略（基于视野分析）

## 第二部分：角色攻略（基于视野信息的最优策略）

### 2.1 平民（Villager）

#### 2.1.1 基础信息
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **私有信息**：自己身份
- **公共信息**：死亡、发言、投票、警长竞选

#### 2.1.2 视野分析
```
优势：无私有信息，不会被针对
劣势：没有任何私有信息，只能通过发言和投票判断
关键洞察：平民的发言空间最大，因为没有信息需要隐藏
```

#### 2.1.3 最优策略

| 场景 | 策略 | 收益分析 |
|------|------|----------|
| 常规局 | 表水 + 找狼 + 投票 | 最大化好人团队收益 |
| 警上 | 诈身份（跳预言家试反应后退水） | 试探对跳者身份，获取信息 |
| 被查杀 | 表水 + 找狼 + 投票 | 平民被查杀没有损失，继续找狼 |
| 情侣局 | 根据情侣发言判断阵营，暗中配合 | 根据情侣身份决定站队 |

#### 2.1.4 策略要点
1. **表水优先级最高**：让好人相信你是平民
2. **找狼线索**：发言状态、投票行为、警上表现
3. **不要轻易站边**：等信息足够再做判断
4. **可以适当激进**：平民没有信息需要隐藏

---

### 2.2 预言家（Seer）

#### 2.2.1 基础信息
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **私有信息**：自己身份、查验结果
- **公共信息**：死亡、发言、投票、警长竞选

#### 2.2.2 视野分析
```
优势：有验人信息，可以明确指出狼人
劣势：身份暴露后会成为狼人首要击杀目标
关键洞察：预言家是好人阵营的核心信息源
```

#### 2.2.3 最优策略

| 场景 | 策略 | 收益分析 |
|------|------|----------|
| 常规局 | 上警 + 报查验 + 抢警长 | 最大化信息价值 |
| 报查杀 | 查杀力度大于金水 | 查杀有天然力度 |
| 被悍跳 | 分析发言逻辑 + 团队配合 | 需要好人支持 |
| 情侣局 | 查验情侣确定阵营 | 人狼恋需隐藏身份 |

#### 2.2.4 策略要点
1. **首夜验人优先级**：情侣 > 疑似狼人 > 警上跳的身份
2. **警徽流设计**：留两轮验人计划，警徽给好人传递信息
3. **报验人策略**：查杀报力度，金水报团队
4. **被查杀处理**：分析对跳者发言逻辑，争取好人站边

---

### 2.3 女巫（Witch）

#### 2.3.1 基础信息
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **私有信息**：自己身份、狼人刀人目标、药水量
- **公共信息**：死亡、发言、投票、警长竞选

#### 2.3.2 视野分析
```
优势：知道狼人刀人目标，可以救人或毒人
劣势：用药信息需要隐藏，不能暴露身份
关键洞察：女巫是夜间有主动技能的神职之一（与守卫同为夜间主动技能）
```

#### 2.3.3 最优策略

| 场景 | 策略 | 收益分析 |
|------|------|----------|
| 首夜 | 自救 > 救情侣 > 救预言家 | 保护自己/情侣价值最高 |
| 后续夜 | 救情侣 > 救关键神职 > 救自己 | 情侣存活=自己存活 |
| 毒药使用 | 确认狼人后使用 | 避免毒错好人 |
| 跳身份 | 被查杀时 + 有明确信息 | 需要团队配合时 |
| 情侣局-好人链 | 解药留给情侣和自己 | 救情侣=救自己 |
| 情侣局-人狼恋 | 解药留给情侣和自己 | 救情侣=救自己（第三方胜利） |

#### 2.3.4 策略要点
1. **用药优先级**：自救 > 救情侣 > 救预言家 > 救守卫 > 救其他
2. **解药使用**：情侣存活意味着自己存活，优先留给情侣
3. **毒药使用时机**：确认狼人身份、局势需要
4. **隐藏身份**：不急于跳身份，保持神秘感
5. **与守卫配合**：避免同守同救，可直接沟通

### 2.4 守卫（Guard）

#### 2.4.1 基础信息
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **私有信息**：自己身份、lastGuardTarget（上一晚守护目标）
- **公共信息**：死亡、发言、投票、警长竞选

#### 2.4.2 视野分析
```
优势：可以保护关键人物，夜晚主动性强
劣势：连守限制，不能连续守护同一人
关键洞察：守卫是好人阵营的防御核心，与女巫同为夜间有主动技能的神职
```

#### 2.4.3 最优策略

| 场景 | 策略 | 收益分析 |
|------|------|----------|
| 常规局 | 守护预言家 > 女巫 > 自己 > 猎人 | 保护核心神职 |
| 轮换守护 | 避免连守 + 不可预测 | 打乱狼人刀法 |
| 跳身份后 | 不说守的是谁，可说历程表水 | 保护信息优势 |
| 局势清晰 | 可与女巫直接配合 | 好人强推时 |
| 自守 | 可选择自守提升存活 | 混乱局势 |
| 情侣局-好人链 | 守护情侣 > 守护预言家 | 情侣存活价值高 |
| 情侣局-人狼恋 | 守护情侣（狼人）| 第三方胜利 |

#### 2.4.4 策略要点
1. **守护优先级**：
   - 情侣局：守护情侣（好人链/第三方）> 守护预言家
   - 非情侣局：守护预言家 > 女巫 > 自己 > 猎人
2. **守护模式**：不可预测，让狼人无法预判
3. **跳身份风险**：狼人会知道上一轮守的是谁
4. **与女巫配合**：避免同守同救，可直接沟通

### 2.5 猎人（Hunter）

#### 2.5.1 基础信息
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **私有信息**：自己身份、canShoot（能否开枪）
- **公共信息**：死亡、发言、投票、警长竞选

#### 2.5.2 视野分析
```
优势：死亡时可以带走一人，有威慑力
劣势：被毒杀不能开枪，需要隐藏身份
关键洞察：猎人是好人阵营的威慑力量
```

#### 2.5.3 最优策略

| 场景 | 策略 | 收益分析 |
|------|------|----------|
| 常规局 | 隐藏身份 + 适度展示好人面 | 避免被刀 |
| 死亡时 | 带走疑似狼人 | 最大化收益 |
| 被毒杀 | 不能开枪 | 损失最大 |
| 同守同救 | 不能开枪 | 避免损失 |
| 情侣局 | 通过发言试探情侣身份 | 暗中配合 |

#### 2.5.4 策略要点
1. **隐藏身份**：不轻易跳猎人身份
2. **死亡处理**：被刀/被投可以开枪，被毒不能开枪
3. **开枪选择**：带走发言最像狼的人
4. **发言策略**：可以适度激进，展示好人面

---

### 2.6 白痴（Idiot）

#### 2.6.1 基础信息
- **胜利条件**：狼人全灭 + 第三方全灭（若存在）
- **私有信息**：自己身份、是否翻牌
- **公共信息**：死亡、发言、投票、警长竞选

#### 2.6.2 视野分析
```
优势：被投后翻牌免疫，失去投票权但保留发言权
劣势：翻牌后无法投票，狼人可以绕过
关键洞察：白痴是存活能力最强的神职
```

#### 2.6.3 最优策略

| 场景 | 策略 | 收益分析 |
|------|------|----------|
| 翻牌前 | 可以激进 + 适度跳身份 | 引导好人 |
| 被投后 | 翻牌 + 继续发言引导 | 免疫放逐 |
| 翻牌后 | 争取警长 + 引导好人 | 利用发言权 |
| 存活 | 翻牌后存活能力最强 | 长期价值 |

#### 2.6.4 策略要点
1. **翻牌优势**：被投后免疫，狼人需要夜间刀杀
2. **发言价值**：翻牌后保留发言权，可以继续引导
3. **警长利用**：翻牌后没有投票权，传警徽安全
4. **激进策略**：可以适度跳身份吸引火力

### 2.7 丘比特（Cupid）

#### 2.7.1 基础信息
- **胜利条件**：根据情侣类型（好/狼/第三方）
- **私有信息**：自己身份、所有存活玩家列表
- **公共信息**：死亡、发言、投票、警长竞选

#### 2.7.2 视野分析
```
优势：可以决定情侣类型，影响全局走向
劣势：连接后无法控制情侣行为
关键洞察：丘比特是第三方势力的创造者
```

#### 2.7.3 最优策略

| 场景 | 策略 | 收益分析 |
|------|------|----------|
| 连接自己+狼人 | 形成第三方 + 保持低调 | 第三方胜利 |
| 连接好人+狼人 | 人狼恋第三方 + 暗中配合 | 第三方胜利 |
| 连接两个好人 | 好人链 + 配合情侣找狼 | 好人胜利 |
| 连接两个狼人 | 狼人链 + 暗中配合狼人 | 狼人胜利 |
| 观察 | 判断情侣类型 + 调整策略 | 灵活应对 |

#### 2.7.4 策略要点
1. **连接选择**：自己+狼人推荐第三方，人狼恋也推荐第三方
2. **隐藏身份**：不暴露自己是丘比特
3. **观察判断**：通过殉情判断情侣类型
4. **灵活站队**：根据局势选择支持方

### 2.8 狼人（Werewolf）

#### 2.8.1 基础信息
- **胜利条件**：屠边（神职全灭或村民全灭）
- **私有信息**：所有狼人同伴、刀人目标
- **公共信息**：死亡、发言、投票、警长竞选

#### 2.8.2 视野分析
```
优势：夜间可以刀人，有团队配合，知道所有狼队友信息
劣势：需要隐藏身份，不能暴露狼队友信息
关键洞察：狼人信息非常全面，但不能暴露自己信息全面，需要靠演技隐藏
```

#### 2.8.3 最优策略

| 场景 | 策略 | 收益分析 |
|------|------|----------|
| 刀人 | 预言家 > 女巫 > 守卫 > 猎人 > 平民 | 优先击杀神职 |
| 悍跳 | 上警跳预言家 + 抢警长 | 抢夺信息权 |
| 警下投票 | 在警下帮忙投票 + 观察局势 | 团队配合 |
| 隐藏 | 装平民不露多余信息 | 长期存活 |
| 冲锋 | 为狼队友冲锋 | 团队配合 |
| 倒钩 | 站边真预言家 | 获取信任 |

#### 2.8.4 策略要点
1. **刀人优先级**：预言家（信息源）> 女巫（用药）> 守卫（保护）> 猎人 > 平民
2. **上警策略**：首夜刀人时讨论好谁上警谁警下拉票
   - 上警狼人：悍跳预言家抢警长
   - 警下狼人：帮忙投票，观察局势，配合团队
3. **隐藏策略**：
   - 装平民：不暴露队友信息，不刻意否定好人
   - 装神牌：不说自己不知道的信息，状态自然
4. **投票策略**：
   - 顺风局：冲票队友
   - 逆风局：倒钩好人
   - 搅局：栽赃好人，一起冲票

## 第三部分：情侣专题

### 3.1 情侣信息说明

```
丘比特连接后：
- 丘比特：不知道情侣的具体身份
- 情侣：会立即收到通知，知道"谁是情侣"
- 但情侣不知道对方的具体身份
- 情侣不知道丘比特是谁
```

### 3.2 情侣胜利条件

```
情侣胜利条件：
- 好人链：情侣都是好人 → 好人胜利条件
- 狼人链：情侣都是狼人 → 狼人胜利条件
- 人狼恋：一方好人一方狼人 → 第三方胜利条件（情侣或丘比特存活，其他人全死）
```

### 3.3 各类情侣策略

非狼人身份拿到情侣，均需要首先推断情侣对方身份。
预言家则首先查验对方即可得知。
狼人拿到情侣身份则可以立刻得知对方身份。
其他角色需要根据后续轮次的表现时刻推断对方身份，保留人狼恋可能性。

#### 3.3.1 好人链情侣（双方都是好人）

与正常策略没区别，多保护一下对方。

#### 3.3.2 狼人链情侣（双方都是狼人）

| 角色 | 策略 | 目标 |
|------|------|------|
| 狼人情侣 | 可暴露给队友 + 两人隐藏 | 狼人胜利 |

#### 3.3.3 人狼恋情侣（一方好人一方狼人）

| 好人方 | 策略 | 目标 |
|--------|------|------|
| 平民 | 根据狼人情侣发言配合 | 第三方胜利 |
| 预言家 | 隐藏身份 + 暗中配合 | 第三方胜利 |
| 女巫 | 解药互留 + 暗中配合 | 第三方胜利 |
| 守卫 | 守护狼人情侣 + 暗中配合 | 第三方胜利 |
| 猎人 | 暗中配合 + 展示好人面 | 第三方胜利 |

| 狼人方 | 策略 | 目标 |
|--------|------|------|

## 第六部分：总结

### 6.1 各角色核心策略

| 角色 | 核心策略 | 关键要点 |
|------|----------|----------|
| 平民 | 表水 + 找狼 | 最大化好人团队 |
| 预言家 | 上警 + 报验人 | 抢夺信息权 |
| 女巫 | 用药 + 隐藏 | 解药优先给情侣 |
| 守卫 | 守护 + 轮换 | 情侣局守护情侣 |
| 猎人 | 隐藏 + 开枪 | 威慑力量 |
| 白痴 | 翻牌 + 发言 | 存活最强 |
| 丘比特 | 连接 + 观察 | 创造第三方 |
| 狼人 | 刀人 + 隐藏 | 夜间优势 |

### 6.2 胜利条件优先级

```
好人：狼人全灭 > 第三方全灭
狼人：屠边（神职/平民）> 第三方全灭
第三方：情侣或丘比特存活 > 其他人全死
```

### 6.3 关键决策点

1. **警长竞选**：预言家/部分狼人上警，其他角色隐藏
2. **夜间用药**：女巫自救/救情侣 > 救预言家
3. **夜间守护**：情侣局守护情侣，非情侣局守护预言家
4. **情侣处理**：人狼恋 = 第三方，目标第三方胜利
5. **投票决策**：信息充分时果断，信息不足时观察
6. **遗言价值**：首夜死亡/白天被投有遗言，第二夜及之后夜晚死亡无遗言
